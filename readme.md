# Разработка плана повышения retention

В данной работе проводится анализ пользователей платформы IT Resume.
В PostgreSQL рассчитаны N-day и Rolling Retention, на их основании построен дашборд в Metabase.
Подробные расчеты с рекомендациями можно посмотреть в документе "План повышения retention платформы IT Resume".
